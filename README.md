# GetItDone

![Python](https://img.shields.io/badge/Python-3.14-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)

A modern, elegant desktop task manager built with Python and CustomTkinter. GetItDone helps you stay organized with a clean interface and task storage.


## Features

-  **Add Tasks** - Quickly add tasks with automatic timestamps
-  **Edit Tasks** - Modify existing tasks via a clean dialog interface
-  **Delete Tasks** - Remove individual tasks with a single click
-  **Storage** - Tasks are saved to JSON automatically
-  **Modern UI** - Beautiful interface 
-  **Theme Toggle** - Switch between light and dark mode with the button in the bottom left corner
-  **Responsive Design** - Clean layout that adapts to window resizing
-  **Keyboard Shortcuts** - Press Enter to add tasks, Escape to cancel edits

## Installation

### Download Executable

Download the latest `GetItDone.exe` from the [Releases](https://github.com/Apollo-WasTaken/GetItDone/releases/tag/v1.0.0) page and run it directly. No installation required.


## Usage

1. **Add a Task**: Type your task in the input field and press Enter or click "Add Task"
2. **Edit a Task**: Click the "Edit" button on any task to modify it
3. **Delete a Task**: Click the "Delete" button on any task to remove it
4. **View Timestamps**: Each task displays when it was created
